# 🔗 My Links Page

A beautiful, interactive links page similar to Linktree. Built with pure HTML, CSS, and JavaScript.

![Demo](images/demo.jpg)

## 🚀 Live Demo

[View Live Site](https://Itsme-sudo.github.io/Links/)

## ✨ Features

- 🌓 Dark/Light mode toggle with localStorage persistence
- 📱 Fully responsive design
- 🎨 Smooth animations and hover effects
- 🔗 One-click copy link functionality
- 🖼️ Profile image with hover effects
- ⌨️ Secret Konami code Easter egg!
- ⚡ Fast loading - no external dependencies except Font Awesome

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
